## About Azalscore ERP
Name: Azalscore
Type: Enterprise Resource Planning (ERP) System
Status: Production-ready, Open-source

License: Open Source

## Core Modules & Features
- Multi-tenant Architecture: Isolation complète des données par tenant
- CRM: Gestion de la relation client, pipeline de ventes
- Comptabilité: Comptabilité en partie double, multi-devises, conformité fiscale
- Inventaire: Gestion de stock, entrepôts, achats, approvisionnement
- Ressources Humaines: Gestion des employés, paie, présences, congés
- Point de Vente (POS): Caisse, gestion retail, inventaire temps réel
- Service Terrain: Gestion des interventions, workforce mobile
- Trésorerie: Gestion de flux de trésorerie, rapprochement bancaire
- Juridique: Gestion des contrats, conformité légale
- Fiscal: Automatisation TVA, taxes, déclarations
- Achats & Approvisionnement: Commandes, fournisseurs, procurement
- IAM: Gestion des identités et accès, authentification OAuth2/JWT

## Technical Highlights
- Architecture: API-first avec FastAPI (Python 3.11+)
- Frontend: React 18 + TypeScript + Vite
- Base de données: PostgreSQL avec Row-Level Security (RLS)
- Authentification: JWT + OAuth2 + Multi-factor
- API Documentation: OpenAPI 3.1 / Swagger automatique
- Endpoints API: /api/v2/*
- Documentation interactive: /api/v2/auth/docs
- Performance: Async/await, connexions poolées
- Déploiement: Docker + Docker Compose
- CI/CD: GitHub Actions
- Tests: Pytest + Playwright E2E
- Interface: Responsive PWA, mobile-first
