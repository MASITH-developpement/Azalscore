## Etape 1 : Connexion (2 min)

1. Ouvrez **https://azalscore.com**
2. Cliquez sur **"Se connecter"**
3. Entrez votre **email** et **mot de passe**
4. Si 2FA active : entrez le code a 6 chiffres
5. Vous etes connecte !

## Etape 3 : Vos Premieres Actions (10 min)

### A. Consulter un Client

1. Menu > **CRM** (ou tapez `@` + nom)
2. Cliquez sur un client
3. Explorez les onglets : Infos, Historique, Documents

### B. Creer un Devis

1. Menu > **Devis** > **+ Nouveau**
2. Selectionnez le client
3. Ajoutez des lignes :
   - Cliquez sur **+ Ligne**
   - Choisissez un produit
   - Indiquez la quantite
4. Cliquez sur **Enregistrer**

### C. Envoyer un Document

1. Ouvrez le devis
2. Cliquez sur **Envoyer**
3. Verifiez le destinataire
4. Cliquez sur **Envoyer**

### D. Consulter une Facture

1. Menu > **Factures**
2. Utilisez les filtres (statut, date)
3. Cliquez sur une facture pour les details

## Les 5 Raccourcis Essentiels

| Raccourci | Action |
|-----------|--------|
| **`/`** | Recherche globale |
| **`@nom`** | Rechercher un client |
| **`#num`** | Rechercher un document |
| **`Escape`** | Fermer / Annuler |
| **`Cmd+S`** | Sauvegarder |
