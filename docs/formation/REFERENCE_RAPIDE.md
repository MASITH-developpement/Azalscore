## Operations Courantes

### Creer un Client
```
CRM > + Nouveau > Remplir > Enregistrer
```

### Creer un Devis
```
Devis > + Nouveau > Client > Lignes > Enregistrer/Envoyer
```

### Creer une Facture
```
Factures > + Nouveau > OU > Commande > Generer facture
```

### Recevoir un Paiement
```
Facture > Enregistrer paiement > Montant > Valider
```

### Creer une Commande Achat
```
Achats > + Nouvelle commande > Fournisseur > Lignes > Confirmer
```

### Recevoir des Marchandises
```
Achats > Commande > Reception > Quantites > Valider
```

## En Cas de Probleme

### Erreur de Saisie
1. Cliquez sur Modifier
2. Corrigez
3. Enregistrez

### Document Bloque
1. Verifiez les champs obligatoires (*)
2. Verifiez les totaux
3. Contactez votre responsable si workflow

### Probleme de Connexion
1. Verifiez votre email/mot de passe
2. Essayez "Mot de passe oublie"
3. Attendez 15 min si compte bloque
4. Contactez l'administrateur

### Erreur Systeme
1. Rafraichissez la page (F5)
2. Deconnectez/Reconnectez
3. Essayez un autre navigateur
4. Contactez le support

---

## Bonnes Pratiques

1. **Sauvegardez regulierement** - Meme si auto-save existe
2. **Verifiez avant validation** - Certaines actions sont irreversibles
3. **Utilisez les favoris** - Gagnez du temps
4. **Deconnectez-vous** - Sur ordinateur partage
5. **Signalez les anomalies** - Au support ou admin
