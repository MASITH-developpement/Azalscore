## Video 1.3 : Naviguer dans l'Interface (5 min)

### Script

```
[INTRO]

NARRATEUR:
"Maintenant que vous etes connecte, decouvrons comment naviguer
efficacement dans AZALSCORE."

[ECRAN - Interface complete]

"AZALSCORE propose deux modes d'affichage."

[MODE AZALSCORE]

"Le mode AZALSCORE, avec son menu horizontal, est ideal pour un
acces rapide aux fonctions essentielles.

[MONTRER - Menu horizontal]

Vous y trouvez :
- Le Cockpit pour une vue d'ensemble
- Les Actions rapides pour creer devis, factures...
- La Vue d'ensemble pour acceder aux modules

[TRANSITION]

[MODE ERP]

Le mode ERP affiche un menu lateral complet, parfait pour les
utilisateurs avances.

[MONTRER - Menu lateral]

Les modules sont organises par categories :
- Gestion : clients, devis, factures
- Finance : comptabilite, tresorerie
- Operations : stocks, production
- Et ainsi de suite...

[ACTION - Changer de mode]

Pour basculer entre les modes, cliquez sur votre avatar et selectionnez
'Basculer en mode ERP' ou 'Basculer en mode AZALSCORE'.

[RECHERCHE]

La barre de recherche est votre meilleur ami.

[ACTION - Ouvrir recherche]

Appuyez sur la touche slash '/' ou cliquez dans la barre.

Tapez ce que vous cherchez : un client, un document, un produit...

[ASTUCE]

Astuce : utilisez @ pour chercher un client, # pour un document,
$ pour un produit.

[DEMO - Recherche @Martin]

Par exemple, '@Martin' affiche tous les clients Martin.

[NOTIFICATIONS]

Enfin, surveillez la cloche de notifications pour ne rien manquer."

[OUTRO]
```

### Exercice Pratique
"Connectez-vous et essayez de passer du mode AZALSCORE au mode ERP.
Utilisez la recherche pour trouver un client existant."

- Activite : historique des interactions

[CONTACTS]

[ACTION - Contact]

- Telephone

[INFORMATIONS COMMERCIALES]

Dans l'onglet Commercial :

[ACTION - Onglet commercial]

- Commercial assigne
- Conditions de paiement
- Remise par defaut

# Module 3 : Devis et Facturation

## Video 3.1 : Creer un Devis (8 min)

### Script

```
[INTRO]

NARRATEUR:
"Le devis est souvent le premier document commercial que vous
envoyez a un client. Voyons comment le creer dans AZALSCORE."

[ECRAN - Menu Devis]

"Accedez au module Devis.

[ACTION - + Nouveau devis]

Cliquez sur '+ Nouveau devis'.

[ECRAN - Formulaire devis]

[SELECTION CLIENT]

Commencez par selectionner le client.

[ACTION - Recherche client]

Tapez le nom et selectionnez dans la liste.

Si le client n'existe pas, vous pouvez le creer rapidement.

[INFORMATIONS EN-TETE]

Le numero de devis est genere automatiquement.

[MONTRER - Numero]

Definissez la date de validite. Par defaut, 30 jours.

[ACTION - Date validite]

[AJOUT DE LIGNES]

Passons aux lignes du devis.

[ACTION - + Ajouter ligne]

Cliquez sur 'Ajouter une ligne'.

[SELECTION PRODUIT]

Recherchez le produit ou service.

[ACTION - Recherche produit]

Selectionnez-le. Le prix se remplit automatiquement.

[QUANTITE ET PRIX]

Ajustez la quantite.

[ACTION - Quantite]

Vous pouvez modifier le prix unitaire si necessaire.

[REMISE]

Pour appliquer une remise :

[ACTION - Remise ligne]

- Sur la ligne : remise en % ou en montant
- Sur le total : remise globale

[MONTRER - Calcul automatique]

Les totaux se calculent en temps reel.

[AJOUT AUTRES LIGNES]

Ajoutez d'autres lignes de la meme maniere.

[VERIFICATION]

Verifiez les totaux :

[MONTRER - Recapitulatif]

- Sous-total HT
- Remises
- Total HT
- TVA
- Total TTC

[CONDITIONS]

Ajoutez des conditions particulieres si besoin :

[ACTION - Conditions]

- Conditions de paiement
- Delai de livraison
- Texte libre

[ENREGISTRER]

Cliquez sur 'Enregistrer' pour sauvegarder en brouillon.

[ACTION - Enregistrer]

Vous pouvez revenir le modifier plus tard."

[OUTRO]
```

# Specifications Techniques

## Format Video
- Resolution : 1920x1080 (Full HD)
- Format : MP4 (H.264)
- Audio : AAC, 128 kbps
- Sous-titres : SRT (francais)

## Style Visuel
- Police : System UI / Arial
- Couleurs : Palette AZALSCORE (bleu #2563EB, gris #6B7280)
- Annotations : Cercles rouges, fleches
- Transitions : Fondu 0.5s

## Voix Off
- Voix : Neutre, professionnelle
- Debit : 150 mots/minute
- Pauses : 1s entre sections

## Musique
- Intro/Outro : Jingle AZALSCORE (5s)
- Fond : Ambient discret (optionnel)
